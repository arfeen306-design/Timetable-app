# School Timetable Generator

Desktop application for creating clash-free school timetables. No installation of Python required — just run the app.

---

## How to install

- **Mac:** Unzip the folder you received. You should see a folder named **School Timetable Generator** with an executable inside (same name, no .app).
- **Windows:** Unzip the folder. You should see **School Timetable Generator.exe** and other files in the same folder.

No separate “installer” is needed. Keep the folder where you want (e.g. Applications on Mac, or Desktop).

---

## How to open the app

- **Mac:** Open the **School Timetable Generator** folder, then **double-click** the executable named **School Timetable Generator**.  
  If macOS says the app is from an unidentified developer: **right‑click** the executable → **Open** → **Open** again.
- **Windows:** **Double-click** **School Timetable Generator.exe**.  
  If Windows SmartScreen appears: choose **More info** → **Run anyway** (only if you trust the person who sent you the app).

The app window will open; you do not need to use the terminal or command line.

---

## Where the app stores project files

- The app **does not** store your projects inside the app folder.
- When you **Save** a project, you choose the location and name (e.g. `MySchool.ttb`). You can save to Documents, Desktop, or any folder.
- When you **Open** a project, you choose the `.ttb` file you saved earlier.
- **Recommendation:** Create a folder (e.g. “Timetable projects”) and save all `.ttb` files there so they are easy to find and back up.

## How to report bugs

Please send the following to the person who gave you the app (or to your IT contact):

1. **What you did** — e.g. “Clicked Generate after entering lessons.”
2. **What you expected** — e.g. “Timetable should generate.”
3. **What actually happened** — e.g. “Error message said …” or “App closed.”
4. **Your system** — Mac or Windows, and version (e.g. macOS 14, Windows 11).

If the app shows an error message, include the exact text or a screenshot.

---

## Quick start

1. Open the app (see **How to open the app** above).
2. Click **New project** (or **Create new**).
3. Optional: click **Load Demo Data** on the first screen to try the app with sample data.
4. Go through the steps: School → Subjects → Classes → Classrooms → Teachers → Lessons → Constraints → Generate.
5. On **Generate**, click **Validate Data**, then **Generate Timetable**.
6. Use **Review & Export** to export to Excel or PDF.

For more detail, see any documentation or guides provided with the app.
