# Packaging assets (optional)

- **icon.icns** — Mac app icon. Add a file named `icon.icns` here; the build will use it. Create from a 512×512 or 1024×1024 PNG using `iconutil` or an online converter.
- **icon.ico** — Windows app icon. Add a file named `icon.ico` here (multi-size: 16, 32, 48, 256). Create from PNG using an image editor or online converter.

If these files are missing, the built app uses the default Python/executable icon.
